Sobre mí

Aprendiendo a crear software que ayude a la gente en su día a día.

Soy estudiante de primer año de Desarrollo de Aplicaciones Multiplataforma (DAM) en el Instituto Tecnológico de Barcelona. Mi enfoque no se centra en un solo lenguaje, sino en entender la lógica de programación para resolver problemas de manera eficiente.

Creación de aplicaciones en Android Studio con Kotlin, incluyendo el desarrollo de un juego de Trivial técnico. Conocimientos en API REST y JDBC para la gestión de datos. Experiencia práctica configurando servidores web y NAS con Raspberry Pi, y programación en C++ para proyectos de robótica con Arduino.
