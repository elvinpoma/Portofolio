# Portafolio - Desarrollador de Apps Multiplataforma

Sitio web personal que exhibe mis proyectos como desarrollador especializado en Kotlin, JavaScript y soluciones con Inteligencia Artificial.

## Proyectos Destacados

### Pi Cloud
Infraestructura en la nube con clúster de Raspberry Pi. Incluye servidor Web en DMZ, NAS con RAID y firewall IPFire. Proyecto escolar de redes avanzadas.

**Tecnologías:** Redes, Seguridad, IPFire, RAID, Raspberry Pi

---

### API-List-App
App Android nativa que consume datos de una API REST en tiempo real. Incluye almacenamiento local para soporte offline, sistema de favoritos y modo oscuro.

**Tecnologías:** Kotlin, Jetpack Compose, Retrofit, Room Database, MVVM

**GitHub:** [Repositorio](https://github.com/elvinpoma/API-List-App)

---

### TeeLab
Plataforma fullstack completa para gestionar catálogo de camisetas y pedidos. Incluye API REST en backend y cliente web responsivo con carrito funcional.

**Tecnologías:** Node.js, Express, JavaScript, HTML5, CSS3, REST API

**GitHub:** [Repositorio](https://github.com/elvinpoma/TeeLab)

---

### Trivial-App
Juego de preguntas y respuestas sobre informática con múltiples categorías y niveles de dificultad. Desarrollado completamente en Kotlin con Jetpack Compose.

**Tecnologías:** Kotlin, Jetpack Compose, ViewModel, StateFlow, Coroutines

**GitHub:** [Repositorio](https://github.com/elvinpoma/Trivial-App)

---

### GoogleMaps-API-App
App con Google Maps integrado que permite crear, editar y gestionar marcadores privados. Incluye autenticación con Supabase y captura de imágenes desde cámara o galería.

**Tecnologías:** Kotlin, Jetpack Compose, Google Maps SDK, Supabase, CRUD

**GitHub:** [Repositorio](https://github.com/elvinpoma/GoogleMaps-API-App)

---

### Portafolio Web
Sitio web personal construido con HTML, CSS y JavaScript vanilla. Incluye tema claro/oscuro, animaciones fluidas y arquitectura modular reutilizable.

**Tecnologías:** HTML5, CSS3, JavaScript ES6+, Responsive Design, Accesibilidad

---

### Arduino RC Car
Coche controlado por Bluetooth mediante interfaz RemoteXY. Usa servos, motores DC y pantallas OLED para control en tiempo real.

**Tecnologías:** C++ (Arduino), Bluetooth, RemoteXY, Electrónica, Robótica
