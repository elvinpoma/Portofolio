## Proyectos Destacados

### API-List-App
Aplicación nativa Android desarrollada con Kotlin y Jetpack Compose que consume datos de una API REST en tiempo real, con soporte offline mediante base de datos local.

- Interfaz declarativa con Jetpack Compose
- Consumo de API REST (Rick and Morty API)
- Base de datos local con Room
- Sistema de favoritos persistente
- Modo oscuro nativo
- Vistas flexibles (lista y grid)

**Stack:** Kotlin, Jetpack Compose, Retrofit, Room, MVVM

[Ver proyecto →](https://github.com/elvinpoma/API-List-App)

### TeeLab
Plataforma fullstack completa para gestionar un catálogo de camisetas y pedidos. Incluye API REST en Backend (Node.js + Express) y cliente web responsivo en Frontend.

- API REST con validación y gestión de inventario
- Frontend SPA con HTML, CSS y JavaScript vanilla
- Carrito de compras funcional
- Sistema de pedidos con IDs secuenciales
- Filtrado avanzado de productos
- Búsqueda y ordenamiento por precio

**Stack:** Node.js, Express, JavaScript, HTML5, CSS3

[Ver proyecto →](https://github.com/elvinpoma/TeeLab)

### Trivial-App
Juego de preguntas y respuestas sobre informática con múltiples categorías y niveles de dificultad. Desarrollado completamente en Kotlin con Jetpack Compose.

- Base de datos extensa de preguntas técnicas
- Múltiples categorías y niveles (Principiante, Intermedio, Experto)
- Temporizador dinámico con indicadores visuales
- Sistema de puntuación y compartición de resultados
- Animaciones fluidas y transiciones elegantes
- Arquitectura MVVM con StateFlow

**Stack:** Kotlin, Jetpack Compose, ViewModel, Coroutines, MVVM

[Ver proyecto →](https://github.com/elvinpoma/Trivial-App)

### GoogleMaps-API-App
Aplicación nativa con Google Maps SDK que permite crear, editar y gestionar marcadores privados con autenticación segura mediante Supabase. Incluye captura de imágenes desde cámara o galería.

- Autenticación segura con Supabase
- Google Maps integrado con gestos intuitivos
- CRUD completo de marcadores
- Captura de fotos desde cámara o galería
- Menú drawer para navegación
- Persistencia de sesión segura
- Almacenamiento de imágenes y ubicaciones

**Stack:** Kotlin, Jetpack Compose, Google Maps SDK, Supabase, MVVM

[Ver proyecto →](https://github.com/elvinpoma/GoogleMaps-API-App)

## Instalación y Uso del Portafolio

### Opción 1: Navegador directo
```bash
open index.html
```

### Opción 2: Servidor local (recomendado)
```bash
npm install
npm run dev
```

### Opción 3: Python
```bash
python -m http.server 8000
# Luego abre: http://localhost:8000
```
